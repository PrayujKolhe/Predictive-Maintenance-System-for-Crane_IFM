# 🚧 IO-Link Based Smart Overload & Safety Monitoring System

## 📌 Project Overview

This project presents an **Industry 4.0 based intelligent safety monitoring system** for industrial lifting equipment such as cranes and hoists using **IO-Link smart sensors**, **IFM PLC**, and **CODESYS**.

The system continuously monitors:
- Vibration
- Temperature
- Sensor communication health

It implements:
- Multi-level alarm handling
- Intelligent shutdown logic
- Real-time HMI monitoring
- IO-Link diagnostics
- Safety interlocks

The project improves industrial safety, reliability, and fault detection while reducing nuisance alarms using delay-based logic.

---

## 🎯 Aim

To design and implement an intelligent **IO-Link based safety and fault monitoring system** for industrial lifting equipment using smart sensors and PLC-based control logic.

---

## 🎯 Objectives

- ✅ Real-time vibration and temperature monitoring
- ✅ Multi-level warning & critical shutdown logic
- ✅ IO-Link digital communication integration
- ✅ 5-second delay logic to prevent false alarms
- ✅ Sensor communication fault detection
- ✅ Relay-based safety shutdown system
- ✅ Industry 4.0 ready architecture

---

## 🏭 Industrial Problem Statement

Industrial lifting systems operate under:
- Heavy loads
- Mechanical stress
- High vibration
- Overheating risks

Traditional analog monitoring systems:
- Lack diagnostics
- Cannot detect communication faults
- Cause false shutdowns
- Have poor scalability

This project solves these limitations using:
- IO-Link smart communication
- PLC-based intelligent control
- Real-time diagnostics
