# 🖥️ HMI Development – IO-Link Smart Safety Monitoring System

## 📌 Overview

This repository contains the **HMI (Human Machine Interface)** developed in **CODESYS SP16** for the **IO-Link Based Smart Overload & Safety Monitoring System**.

The HMI provides:
- Real-time sensor monitoring
- Alarm visualization
- System status indication
- Fault notifications
- Operator interaction & reset controls

The interface is designed for industrial safety applications and provides a clean, user-friendly dashboard for monitoring lifting equipment conditions.

---

## 🛠️ Technologies Used

- CODESYS SP16 Visualization
- IFM CR0708S PLC
- Modbus TCP
- IO-Link Communication
- Industrial HMI Design

---

## 🎯 HMI Features

### ✅ Real-Time Monitoring
- Temperature monitoring
- Vibration monitoring
- Live sensor updates every 100 ms

### ✅ Alarm System
- Warning indicators
- Critical shutdown alerts
- Sensor communication fault alarms

### ✅ Safety Interface
- Emergency shutdown status
- Manual reset button
- System health monitoring

### ✅ Visualization Elements
- Status indicators
- Trend graphs
- Alarm popups
- Color-coded warnings
